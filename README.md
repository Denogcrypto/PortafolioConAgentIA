## 🚀 Cómo correrlo localmente

No necesitás instalar nada. Es HTML puro.

```bash
# Cloná el repo
git clone https://github.com/Denogcrypto/portfolio-agent.git

# Abrí el archivo en tu browser
open index.html
```

O simplemente arrastrá el `index.html` al browser.

---

## 🔧 Cómo importar el agente en n8n

1. Abrí n8n → **New Workflow**
2. Menú `⋯` → **Import from file**
3. Seleccioná `dennis-ferraro-agent.json`
4. Conectá tu credencial de **OpenAI**
5. Activá el workflow ✅
